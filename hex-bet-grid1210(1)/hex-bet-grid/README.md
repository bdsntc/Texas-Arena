# AgentPay

**AgentPay** is a Web3-native AI application that demonstrates how autonomous AI agents can pay for intelligence using the x402 protocol.

Instead of traditional API keys or subscriptions, AgentPay visualizes a future where AI agents independently request paid capabilities, execute on-chain payments, and unlock premium intelligence in real time.
